# CVMint – AI Resume Analyzer & Builder

## 🚀 Overview
CVMint is an AI-powered resume analyzer and builder that helps users create optimized resumes with better structure, ATS scoring, and content improvement suggestions.

## ✨ Features
- Resume Builder (multi-step form)
- Resume Analyzer (ATS-style scoring)
- AI-powered content suggestions
- Resume preview & export
- Authentication system (Firebase)
- Dashboard with saved resumes

## 🛠️ Tech Stack
- React.js
- Firebase (Auth + Firestore)
- JavaScript
- CSS / Tailwind (if used)
- OpenAI / Claude / Gemini APIs
